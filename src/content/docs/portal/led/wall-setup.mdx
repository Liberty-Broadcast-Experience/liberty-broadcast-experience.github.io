To build an LED video wall, you only need four things:

- LED panels
- Power cables
- Data cables
- LED video processor

<Steps>
1. Physically assemble the LED wall in the shape that you want.
2. Connect the power cables.
3. Connect the data cables.
4. Map the layout of the video wall with the processor.
</Steps>
